Behnevis Persian Transliteration support for wordpress.

== Description ==

This plugin offers Persian language transliteration support for your wordpress blogs using Behnevis transliteration API. 

== Installation ==

1. Upload `behnevis-transliteration` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
